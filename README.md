# 🌐 Web-Based RAG Chatbot

A powerful RAG (Retrieval-Augmented Generation) chatbot that can answer questions based on any URLs you provide.

## Features
- Load multiple URLs (any website)
- Ask questions based on the content
- Built with LangChain, ChromaDB, and Groq LLM
